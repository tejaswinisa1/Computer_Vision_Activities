# Computer Vision Activities

A comprehensive collection of Jupyter notebooks covering various computer vision concepts, techniques, and applications in deep learning.

## 📚 Repository Contents

### Core Computer Vision Topics

#### **1. GrayScale Image Processing** (`__GrayScale__.ipynb`)
- Image conversion to grayscale format
- Grayscale image manipulation techniques
- Practical applications of grayscale processing in computer vision

#### **2. Object Counting** (`__OBJECT_COUNT__.ipynb`)
- Techniques for detecting and counting objects in images
- Object detection algorithms
- Counting methodologies for automated image analysis

#### **3. RMSE and MSE** (`__RMSE_and_MSE__.ipynb`)
- Root Mean Square Error (RMSE) calculations
- Mean Square Error (MSE) analysis
- Image quality assessment metrics
- Error analysis in computer vision tasks

#### **4. SAM2 and SLAM** (`__SAM2_and_SLAM_.ipynb`)
- Segment Anything Model 2 (SAM2) implementation
- Simultaneous Localization and Mapping (SLAM)
- Advanced segmentation and localization techniques

#### **5. Deep Learning Neural Networks** (`__VGG_and_RESNET__.ipynb`)
- VGG (Visual Geometry Group) architecture
- ResNet (Residual Networks) implementation
- Convolutional neural network fundamentals
- Deep learning models for image classification

#### **6. Tutorial 7** (`__TUTORIAL-7__.ipynb`)
- Comprehensive tutorial covering various computer vision techniques
- Practical examples and exercises

### Additional Resources

#### **7. Computer Vision in Deep Learning** (`computervision _in _DL.ipynb`)
- Introduction to computer vision applications in deep learning
- Overview of key concepts and methodologies

#### **8. TensorFlow to TFLite Conversion** (`tensorflow _to_tflite.ipynb`)
- Converting TensorFlow models to TensorFlow Lite format
- Model optimization for mobile and edge devices
- Deployment considerations for lightweight models

#### **9. Week 1 Activity** (`week1_activity.ipynb`)
- Introductory activity for week 1
- Foundational concepts in computer vision

## 📖 How to Use

1. **Clone the repository:**
   ```bash
   git clone https://github.com/tejaswinisa1/Computer_Vision_Activities.git
   cd Computer_Vision_Activities
   ```

2. **Install required dependencies:**
   ```bash
   pip install jupyter tensorflow opencv-python numpy matplotlib seaborn
   ```

3. **Launch Jupyter Notebook:**
   ```bash
   jupyter notebook
   ```

4. **Open any notebook** and run cells to explore the computer vision concepts and implementations.

## 🎯 Learning Path

**Recommended order of exploration:**
1. Start with `week1_activity.ipynb` - Basic concepts
2. Move to `computervision _in _DL.ipynb` - Overview of the field
